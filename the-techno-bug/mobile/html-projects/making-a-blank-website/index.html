Tag Definitions

<!DOCTYPE html>: This is a DTD ( Document type declaration ). It tells the browser that this is an HTML5 document.

<html> and </html>: These are the opening and closing tags for all html code

<head> and </head>: These are the opening and closing tags for the head( the tab icon for browsers )

<title> and </title>: These tags define the title ( what shows in the tab icon )

<body> and </body>: These are the opening and closing tags for the body ( what shows on the actual website )
