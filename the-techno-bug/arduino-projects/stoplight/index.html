int red = 9;
int ylw = 10;
int grn = 11;

void setup () {
   // put your setup code here, to run once:
   pinMode ( red ,   OUTPUT );
   pinMode ( ylw ,   OUTPUT );
   pinMode ( grn ,   OUTPUT );
}

void loop () {
   // put your main code here, to run repeatedly:
   digitalWrite (red ,   HIGH );
   digitalWrite (ylw ,   LOW );
   digitalWrite (grn ,   LOW );
   delay (20000);   //ms
   digitalWrite (red ,   LOW );
   digitalWrite (ylw ,   LOW );
   digitalWrite (grn ,   HIGH );
   delay (20000);   //ms
}

Connections

  1. Connect LED with resistor to Arduino Pin 13 and Connect Short leg to Negative Rail
  2. Connect Tact Switch with one pin to Negative Rail and other pin to Arduino Pin 10
  3. Connect Negative Rail to GND on the Arduino

Function definitions

//Blink Sketch - Blink Arduino's onboard LED : The // indicates that the line is a comment, something for the programmer to read instead of the computer.

void setup () {}: void setup() is a way of telling the arduino that this is what I want you to run once ( Kind of like setting up your computer - you only have to do it once! ) The curly brackets indicate what is inside void setup() and what is not.

pinMode (pin, mode) : The pinMode function tells the arduino to set up the pin to mode.

LED_BUILTIN : This is the pin that represents the onboard LED.

OUTPUT : One of the modes from pinMode.

void loop () {}: void loop() is a way of telling the arduino that this is what I want you to run until someone resets or turns off the arduino ( like booting your computer: It keeps running until you shut it down! ) The curly brackets indicate what is inside void setup() and what is not.

digitalWrite (pin, status): The digitalWrite function turns the pin to positive or negative, depending on the status.

HIGH : This represents positive energy in the digitalWrite function.

LOW : This represents negative energy in the digitalWrite function.

delay (milliseconds): The delay function pauses the arduino for the number of milliseconds.

Last but not least, don't forget the semicolon (";") at the end of each line! The semicolon tells the arduino that one instruction is over.
