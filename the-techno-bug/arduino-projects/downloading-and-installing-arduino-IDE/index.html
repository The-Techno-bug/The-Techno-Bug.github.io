Downloading the Arduino IDE

Download instructions for Windows 10

  1. Go on the Arduino website.
  2. Click Software > Downloads.
  3. Click the Windows Installer button.
  4. You can click Just Download or Contribute and Download.
  5. After the installer downloads, open file explorer
  6. Double-Click the installer.
  7. There will be several prompts. Click next to all of them.
  8. Last, click install to install the USB drivers.
  9. That is it for installing Arduino IDE on Windows 10.

Download instructions for Linux

  1. Open Linux terminal.
  2. Type "sudo apt-get install arduino".
  3. Enter your password when you are prompted to do so
  4. Enter "y" at the next prompt
  5. That is it for installing Arduino IDE on Linux.
