Tag Definitions

<html></html>: This tag starts/ends the HTML ( The <html> starts the HTML and the </html> ends the HTML ) <head></head>: This tag starts/ends the head of the webpage ( The Head is the tab or top of the webpage and can also be used for CSS and Javascript ) <body></body>: This tag starts/ends the body of the webpage ( The body is the main content of the page ) <title></title>: This tag starts/ends the title of the webpage ( The Title is the name that appears in the tab icon ) <p></p>: This tag starts/ends a paragraph <b></b>: This tag makes any text inside it Bold <i></i>: This tag makes any text inside it Italic <strong></strong>: This tag is the same as the <b></b> tag but screen readers say the word/words inside it strongly <em></em>: This tag is the same as the <i></i> tag but screen readers say the word/words inside it with emphasis
