Tag Definitions

<DOCTYPE html>: This is a document type declaration, telling the running browser that this is an HTML5 Document <html></html>: This tag starts/ends the HTML ( The <html> starts the HTML and the </html> ends the HTML ) <head></head>: This tag starts/ends the head of the webpage ( The Head is the tab or top of the webpage and can also be used for CSS and Javascript ) <body></body>: This tag starts/ends the body of the webpage ( The body is the main content of the page ) <title></title>: This tag starts/ends the title of the webpage ( The Title is the name that appears in the tab icon )
