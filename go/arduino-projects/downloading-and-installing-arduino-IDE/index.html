Downloading the Arduino IDE

Downloading Arduino IDE on Windows

  1. Go on the Arduino Website and click the Windows 7 and newer button
  2. You can click just download or contribute and download. I have clicked just download.
  3. Open file explorer
  4. Double-click the windows installer file
  5. Follow the instructions on the screen ( for detailed instructions, click here ).

Downloading Arduino IDE on Linux

  1. Open Linux Terminal
  2. Type sudo apt-get install arduino into the linux terminal
  3. Type y when the terminal asks for confirmation
  4. Wait for the arduino IDE to download

That is how you download the arduino IDE on windows
